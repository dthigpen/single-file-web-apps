# Single-File Web Apps

A collection of portable, privacy-respecting, local-first web apps. Each application is just a single `.html` file that you can download and click yourself.

Check out the docs for more info: **[https://dthigpen.github.io/single-file-web-apps/](https://dthigpen.github.io/single-file-web-apps/)**

## Motivation

I switch to a "dumbphone" in 2026 which was pretty locked down. I can however transfer `.html` files to it and open them in the web browser, providing app like behavior.
